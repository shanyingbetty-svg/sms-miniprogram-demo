# SMS 小程序预览指南 🚀

## 📱 如何查看效果

**方法一：本地运行**
1. 打开浏览器访问：http://localhost:3000
2. 确保终端中 npm start 进程正在运行

**方法二：手机预览**
- 如果在同一个网络下，可以通过局域网IP访问
- 建议在开发者工具中选择 "Mobile" 或 "iPhone" 设备模式

## 🎯 核心界面预览

### 🏠 首页 (Home Screen)
**功能特点：**
- 顶部：用户身份显示 "社区物资管理员-张华 | 幸福社区储备点"
- 中部：核心数据看板（总物资数: 120件 | 待巡检: 2项 | 异常/临期: 1项）
- 底部：功能模块网格（日常物资巡检维护 ✅ | 就近互助调度 ✅）

**交互：**
- 点击 "日常物资巡检维护" → 跳转到巡检列表页
- 点击 "就近互助调度" → 跳转到互助调度查询页

### 📋 巡检列表页 (Inspection List)
**功能特点：**
- 搜索框：支持输入物资名称检索
- 标签切换：全部台账 | 待巡检 (2) | 临期/异常 (1)
- 默认显示待巡检标签

**卡片1 - 待巡检物资：**
- 标题：防汛物资 - 便携式抽水泵
- 规格：50L/min | 共 2 台 | 存放于 1 号储备柜
- 状态：[待巡检] (黄色标签)
- 按钮：[去巡检打卡] → 跳转到巡检表单页

**卡片2 - 临期物资：**
- 标题：消防物资 - ABC干粉灭火器
- 规格：4kg | 共 15 瓶 | 存放于 消防物资间
- 状态：[临期预警] (红色标签) + "2026-08到期"
- 按钮：[一键维护] → 跳转到维护登记页

### 📝 巡检表单页 (Inspection Form)
**功能特点：**
- 顶部：返回按钮 + "物资巡检打卡 - 便携式抽水泵" 标题
- 物资基础信息区域（只读显示）
- 巡检结果选择：完好(默认)、破损、受潮、过期
- 现场照片上传：点击显示 "照片已上传" 状态
- 备注输入框：支持多行文本输入
- 底部按钮：一键提交巡检记录

**交互流程：**
- 点击照片上传 → 模拟打开相册 → 显示上传成功
- 提交后 → Toast提示 "提交成功，已自动生成维护台账！" → 返回列表页（状态变为正常）

### 🔧 维护登记页 (Maintenance)
**功能特点：**
- 顶部：返回按钮 + "临期/异常物资维护" 标题
- 红色警告横幅：显示异常物资提醒
- 待维护物资卡片：ABC干粉灭火器（红色预警边框）
- 维护方式下拉选择：更换气瓶/药剂、外观除锈、移位防潮
- 责任人输入框（默认：张华）
- 底部按钮：登记维护记录

**交互流程：**
- 点击登记 → Dialog提示 "维护台账登记成功，已上报镇级管理端督办" → 确定返回

### 🗺️ 互助调度查询页 (Mutual Aid Search)
**功能特点：**
- 顶部：就近互助调度标题
- 需求横幅：红色警告显示 "当前本社区：抢险工具-铁锹 库存不足！缺口：30把"
- 标签切换：地图模式 | 列表模式（默认列表）

**地图模式：**
- 模拟地图背景 + 三个社区定位点（朝阳、向阳、幸福社区）
- 下方显示周边物资列表

**列表模式：**
- 朝阳社区储备点（距离 1.2km）
  - 铁锹（富余 50 把）、防毒面具（富余 20 个）
  - [发起调度] 按钮
- 向阳社区储备点（距离 2.5km）
  - 铁锹（富余 20 把）
  - [发起调度] 按钮

### 📤 调度申请页 (Dispatch Request)
**功能特点：**
- 顶部：返回按钮 + "发起跨社区调度申请" 标题
- 调度信息卡片：展示调出社区、距离、物资详情
- 申请数量控制：
  - 减号按钮 ← [30] → 加号按钮
  - 限制：1-50 把（基于富余数量）
- 紧急程度选择：特急(红)、紧急(黄)、一般(蓝)
- 调度原因：默认 "应对今晚突发强降雨抢险"
- 底部按钮：确认发起调度申请

**交互流程：**
- 点击提交 → Loading动画 "平台正在智能测算最优运输路径..."
- 2秒后 → 成功提示 "申请已发送，等待对方社区接单" → 跳转到跟踪页

### 📦 进度跟踪签收页 (Tracking & Sign-off)
**功能特点：**
- 顶部：大字体 "物资配送中" 状态
- 进度步骤条（垂直展示）：
  1. ✓ 已发起：幸福社区于 21:45 发起申请
  2. ✓ 已接单：朝阳社区已确认并出库
  3. 🔄 配送中：配送员正在前往...(当前进行中)
  4. ⚪ 待签收：签收核验(置灰)

**配送信息显示：**
- 配送师傅：李师傅
- 运输车辆：鲁B12345
- 配送物资：30把铁锹
- 预计到达：22:30

**签收区域：**
- 3秒后自动激活签收功能
- 勾选确认框："我已核验数量及物资完好度"
- 激活 [线上确认签收] 按钮

**完成流程：**
- 点击签收 → 全屏成功动效 "签收成功！已完成调度闭环"
- 提示："系统已自动生成跨社区有偿结算单据，纳入月度费用核算"
- 完成后返回首页

## 🎨 视觉设计亮点

### 颜色系统
```
主色调：应急救援蓝 #1B59F8
正常状态：绿色 #52C41A
警告状态：黄色 #FAAD14
危险状态：红色 #FF4D4F
```

### 交互细节
- 按钮悬停动效和阴影效果
- 渐变背景设计
- 圆角卡片和现代化边框
- 状态颜色一致性

### 移动端优化
- 固定底部操作栏
- 触摸友好的按钮大小
- 滚动区域优化
- 响应式布局

## 🔄 完整使用流程

**巡检流程：** 首页 → 巡检列表 → 巡检表单 → 提交成功
**调度流程：** 首页 → 互助调度 → 调度申请 → 进度跟踪 → 签收完成

所有页面都支持返回导航，状态流转清晰，用户体验流畅！
